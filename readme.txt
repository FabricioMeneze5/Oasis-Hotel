English:

Oasis Hotel

Project developed as part of my portfolio during my time at the British School of Creative Arts - EBAC. This is a fictitious website for a hotel, spa, and gym, created using Bootstrap, jQuery, CSS, and HTML.


--Description--
This project is a responsive website that simulates a fictitious hotel, spa, and gym, emphasizing user experience. It was developed as a way to showcase my skills in front-end and web design.


--Technologies Used--
	HTML
	CSS
	bootstrap
	jQuery


--Key Features--
Responsiveness: The site is fully responsive, adapting to different devices to ensure a consistent user experience.
Attractive Design: I utilized Bootstrap to create an appealing and functional layout, keeping aesthetics and usability in mind.
Smooth Interaction: The use of jQuery adds smooth and pleasant interactions for the user while navigating the site.

Português:

Oasis Hotel

Projeto desenvolvido como parte do meu portfólio durante meu período na Escola Britânica de Artes Criativas - EBAC. Este é um site fictício para um hotel, spa e academia, criado utilizando Bootstrap, jQuery, CSS e HTML.

--Descrição--
Este projeto é um site responsivo que simula um hotel, spa e academia fictícios, com ênfase na experiência do usuário. Foi desenvolvido como uma maneira de demonstrar minhas habilidades em front-end e design web.


--Tecnologias Utilizadas--
	HTML
	CSS
	Bootstrap
	jQuery


--Funcionalidades Principais--
Responsividade: O site é totalmente responsivo, adaptando-se a diferentes dispositivos para garantir uma experiência consistente ao usuário.
Design Atraente: Utilizei o Bootstrap para criar um layout atrativo e funcional, mantendo a estética e a usabilidade em mente.
Interação Suave: O uso do jQuery adiciona interações suaves e agradáveis para o usuário durante a navegação no site.
